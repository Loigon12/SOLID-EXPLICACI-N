El Principio Abierto/Cerrado (OCP)

Definición: Las entidades de software deben estar abiertas para su extensión, pero cerradas para su modificación.

Ejemplo Incorrecto (Violando OCP)

En este ejemplo, si queremos agregar una nueva forma (como un Triángulo), tendríamos que entrar a modificar la clase CalculadoraDeAreas agregando más condicionales. Esto rompe el principio porque no está "cerrada a la modificación".


class CalculadoraDeAreas {
    calcularArea(forma) {
        if (forma.tipo === 'circulo') {
            return Math.PI * forma.radio * forma.radio;
        } else if (forma.tipo === 'cuadrado') {
            return forma.lado * forma.lado;
        }
        // ¡PELIGRO! Si agregamos un triángulo, hay que modificar esta clase.
    }
}

Ejemplo Correcto (Aplicando OCP)

Aquí, delegamos la responsabilidad de calcular el área a cada clase específica. Ahora la CalculadoraDeAreas está "cerrada" a modificaciones. Si queremos agregar un Triángulo, simplemente creamos una nueva clase que extienda de Forma. ¡La calculadora no se toca!


// Interfaz o clase base
class Forma {
    calcularArea() {
        throw new Error("El método calcularArea() debe ser implementado");
    }
}

// Extensión 1: Círculo
class Circulo extends Forma {
    constructor(radio) {
        super();
        this.radio = radio;
    }
    calcularArea() {
        return Math.PI * this.radio * this.radio;
    }
}

// Extensión 2: Cuadrado
class Cuadrado extends Forma {
    constructor(lado) {
        super();
        this.lado = lado;
    }
    calcularArea() {
        return this.lado * this.lado;
    }
}

// La clase que utiliza las formas
class CalculadoraDeAreas {
    obtenerArea(forma) {
        // No le importa qué forma es, solo llama a calcularArea()
        return forma.calcularArea(); 
    }
}

// ¡Para agregar un Triángulo solo creamos una clase nueva!
class Triangulo extends Forma {
    constructor(base, altura) {
        super();
        this.base = base;
        this.altura = altura;
    }
    calcularArea() {
        return (this.base * this.altura) / 2;
    }
}
